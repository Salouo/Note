## 1. Upload my branch

1. Switch to `chen` branch.

   ```sh
   git checkout chen
   ```

   

2. Check status of modifications

   ```sh
   git status
   ```

   

3. Stage the modifications of `chen/`.

   ```sh
   git add chen/
   ```



4. Commit a comment.

   ```sh
   git commit -m 'update'
   ```



5. Push the local submission to remote repository.

   ```sh
   git push origin chen
   ```
